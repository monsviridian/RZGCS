RZ Ground Control Station 
========================= 
 
Installation: 
1. Stellen Sie sicher, dass Python 3.8 oder höher installiert ist. 
2. Führen Sie 'install.bat' aus, um alle erforderlichen Abhängigkeiten zu installieren. 
3. Führen Sie 'start_rzgcs.bat' aus, um die Anwendung zu starten. 
4. Optional: Führen Sie 'create_shortcut.bat' aus, um eine Desktop-Verknüpfung zu erstellen. 
 
Systemanforderungen: 
- Windows 10/11, macOS 10.14+, Ubuntu 20.04+, oder Raspberry Pi OS 
- Python 3.8 oder höher 
- 2GB RAM (4GB empfohlen)
